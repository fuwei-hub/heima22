河池学院学生会改革实施方案

为深入贯彻习近平新时代中国特色社会主义思想特别是习近平总书记关于青年工作的重要思想， 全面落实共青团中央、教育部、全国学联制定的《关于推动高校学生会（研究生会）深化改革的若干意见》， 使学生会有限的资源得以合理分配，促进学生会更健康的发展，精简学生会的架构，现做出了对部分部门合并的决定， 并将合并之后的学生会的各部门工作范围进行修订.

（一）指导思想

全面贯彻习近平新时代中国特色社会主义思想和党的十九大精神， 认真贯彻习近平总书记系列重要讲话精神特别是对青年学生工作的重要指示， 使得我校学生会在学校党委的领导下、学校团委的具体指导下以保持和增强政治性、先进性、群众性为目标， 坚持立德树人，紧扣时代主题，突出问题导向，创新体制机制，强化学生会“自我教育、自我管理、自我服务、 自我监督”的职能，激发动力、提升活力、增强吸引力和凝聚力，使学生会更好地代表和服务广 大学生， 更好地团结和凝聚广大学生听党话、跟党走，带领我校广大同学秉承“修德、弘毅、勤业、致远”的校训， 为把我校建设成为特色鲜明、基础扎实、优势凸显的区域知名地方应用型大学和实现中华民族伟大复兴中国梦的奋斗目标汇聚磅礴的青春力量。

（二）主要目标

  1. 作用发挥更加充分。积极争取我校各级党组织和团组织的支持帮助，使我校学生会职能定位更加明确，工作自主性、规范性进一步增强，联系代表、服务同学和维护同学权益的作用发挥更加充分，在参与我校建设中的作用显著提升.
  2. 代表性更加广泛。进一步扩大我校学生会的代表性，规范学生代表大会代表名额分配比例和产生方式，重点扩大普通学生代表的比例，真正选出品学兼优的学生代表、学生干部；建立学生代表、学生干部直接联系同学、听取意见的制度。
  3. 队伍作风更加严实。进一步规范各级学生会领导机构的产生方式；规范学生会干部的选拔标准和评价机制，建立健全学生干部退出机制；推进学生干部队伍作风建设，不断强化学生干部的群众意识、责任意识、奉献意识，以实际行动做好广大学生的表率，赢得广大学生的信赖。
  4. 工作效能更加彰显。进一步明确学校团学组织格局，理顺我校学生会与其他学生组织的关系；规范校院两级学生会的机构设置，减少层级、提升效能，充分发挥互联网作用，构建扁平高效的组织体系；进一步推动建设“学校、学院、班级” 三级联动的工作格局，推动校院两级学生会进一步完善广大学 生参与、监督和评议的体制机制，努力建设学习型、服务型、创新型学生会，不断提高引领和服务广大学生的水平和能力。

（三）改革举措

  1. 改革健全学生会的领导体制和工作机制
  2. 改革优化学生会的组织机构和功能定位
  3. 改革完善学生会成员的选拔
  4. 改革创新学生会的工作方式和工作方法
